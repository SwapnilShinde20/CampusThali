CampusThali

Simplifying Food. Empowering Campuses.

CampusThali is a smart, student-first food platform designed to solve one of the most common problems in college campuses — access to affordable, reliable, and transparent food options.

It connects campus kitchens directly with students through a seamless, intuitive, and scalable digital experience.

🧠 Problem

College students face daily challenges when it comes to food:

Long waiting queues
Limited and unclear food options
No pricing transparency
Inefficient manual systems

This results in wasted time, poor experience, and lack of trust.

💡 Solution

CampusThali provides a centralized digital platform where:

Students can browse meals easily
Kitchens can manage menus dynamically
Orders are streamlined and efficient

It transforms campus food into a fast, transparent, and scalable ecosystem.

🧪 How to Run Locally
# Clone the repository
git clone https://github.com/Prakshal97/campus-thali-kitchen

# Go into the project folder
cd campus-thali-kitchen

# Install dependencies
npm install

# Start development server
npm run dev
